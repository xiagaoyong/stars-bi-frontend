## Environment Prepare

Install `node_modules`:

```bash
npm install
```

or

```bash
yarn
```

Ant Design Pro 提供一些有用的脚本来帮助您快速启动和构建Web项目、代码风格检查和测试。

### Start project

```bash
npm start
```
